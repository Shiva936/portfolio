# Shiv Deepak Sharma — Backend Engineering Portfolio

This repository contains a modern, responsive portfolio website built with **React + Vite + Tailwind CSS**.

## Development

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Build

```bash
npm run build
```

The production-ready static output will be generated into `dist/`.

## Deployment

The project is configured with a relative base path (`./`) so the build output is compatible with GitHub Pages or other static hosting providers.

### GitHub Pages

Use the deploy script to publish the `dist/` folder to GitHub Pages:

```bash
npm run deploy
```
